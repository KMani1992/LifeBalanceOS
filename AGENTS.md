# AI Agents Configuration

This document outlines the AI agents and their configurations used in the LifeBalanceOS project.

## Overview

LifeBalanceOS uses AI agents to help with:
- Daily planning and scheduling
- Goal tracking and analysis
- Financial advice
- Family organization
- Personal reflections

## Agent Types

### Daily Agent
- Manages daily tasks and checklist
- Provides daily recommendations

### Goal Agent
- Helps set and track goals
- Generates progress reports

### Finance Agent
- Tracks financial metrics
- Provides financial insights

### Family Agent
- Manages kids' activities
- Family calendar coordination

### Reflection Agent
- Processes weekly reflections
- Generates insights

## Configuration

Configure agents via environment variables in `.env.local`.
